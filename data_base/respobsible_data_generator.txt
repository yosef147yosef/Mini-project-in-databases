Attribute: Worker_ID
Command: =INDEX(Worker_ID.csv!$A:$A, RANDBETWEEN(2,COUNTA(Worker_ID.csv!$A:$A)))
Description: Selects a random worker ID from the Worker_ID.csv file for each deposit.

Attribute: Deposit_ID
Command: =INDEX(Deposit_ID.csv!$A:$A, RANDBETWEEN(2,COUNTA(Deposit_ID.csv!$A:$A)))
Description: Selects a random deposit ID from the Deposit_ID.csv file for each worker.
